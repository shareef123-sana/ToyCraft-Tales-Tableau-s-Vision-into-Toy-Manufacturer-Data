## 📌 Project Overview
This project demonstrates **data visualization and analysis** using Tableau.  
It analyzes sales data for a toy manufacturing company, highlighting category performance, revenue trends, and discount impact.

---

## 🗂️ Project Features
- Interactive Tableau dashboards  
- Category-wise sales analysis  
- Monthly revenue trends visualization  
- Discount impact on revenue  
- Highlighting top-performing products  

---

## 📊 Key Insights
- Identify best-selling toy categories  
- Track sales trends over time  
- Evaluate impact of discount strategies on revenue  
- Understand customer purchase behavior  

---

## 🛠️ Tools Used
- Tableau Desktop / Tableau Public  
- Sample sales dataset  
- GitHub for version control  

---

## 🎯 Purpose
This project demonstrates **data visualization skills, business analysis thinking, and ability to convert raw data into actionable insights**.  
It is portfolio-ready for internships or entry-level Data Analyst roles.
